# Content Standards

> Guidelines for creating and maintaining content on Roma Mart

## Overview

This guide covers content standards for text, images, and media across the Roma Mart website.

## Writing Style

### Tone

- Professional yet approachable
- Clear and concise
- Action-oriented
- Inclusive language

### Formatting

- Use sentence case for headings
- Keep paragraphs short (2-3 sentences)
- Use bullet points for lists
- Include clear calls-to-action

## Brand Elements

### Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Navy | #020178 | Primary brand color |
| Yellow | #E4B340 | Accents, CTAs |
| White | #FFFFFF | Backgrounds |

### Typography

- **Headings:** Outfit (600 weight)
- **Body:** Inter (400 weight)
- **Sizes:** Use design token scale

## Image Guidelines

### Requirements

- All images must have `alt` text
- Use WebP format when possible
- Compress images before upload
- Maintain aspect ratios

### Dimensions

| Usage | Dimensions | Format |
|-------|------------|--------|
| Hero | 1920x1080 | WebP/JPG |
| Product | 800x800 | WebP/PNG |
| Thumbnail | 400x300 | WebP/JPG |
| Logo | SVG | SVG |

### Photography Style

- Well-lit, natural lighting preferred
- Clean backgrounds
- Focus on products/services
- Represent diverse customers

## Content Types

### Service Descriptions

- Clear benefit statement
- Key features (3-5 bullets)
- Call-to-action

### Location Information

- Address with postal code
- Operating hours
- Services available
- Google Maps integration

### Menu Items (RoCaf√©)

- Item name
- Description (1-2 sentences)
- Price
- Category

## Accessibility

- Alt text must describe image content
- Don't use images of text
- Ensure sufficient color contrast
- Provide text alternatives
